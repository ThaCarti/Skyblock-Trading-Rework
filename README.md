# Skyblock Progression Villagers

A Minecraft datapack that restores progression paths for Skyblock and void-world survival through custom villager and wandering trader trades.

## Overview

Many items, blocks, and gameplay systems become inaccessible in traditional Skyblock worlds due to the lack of normal world generation. This datapack addresses those progression gaps by expanding and rebalancing villager trades while maintaining a vanilla-friendly survival experience.

The goal is not to make Skyblock easier, but to make it fully completable without relying on commands, creative mode, or external modifications.

## Features

### Villager Trade Overhauls

Custom trade tables have been created for multiple villager professions, including:

* Librarian
* Leatherworker
* Mason
* Shepherd
* Toolsmith
* Weaponsmith
* Wandering Trader

### Progression Restoration

Provides renewable access to many resources that are normally difficult or impossible to obtain in Skyblock worlds, including:

* Sand
* Red Sand
* Clay
* Terracotta
* Calcite
* Tuff
* Deepslate
* Basalt
* Mud
* Gilded Blackstone
* Frogspawn
* Turtle Eggs
* Sculk Catalysts
* Sponge
* Allays
* Sniffers
* Rare plants and saplings
* Music Discs
* And many more

### Vanilla-Friendly Design

Trades are intentionally balanced to preserve progression and encourage normal gameplay.

Players are still expected to:

* Build farms
* Gather resources
* Level villagers
* Invest emeralds

The datapack provides progression paths rather than free resources.

## Installation

1. Download the latest release.

2. Place **SBTR.zip** into the **datapacks** folder of your Minecraft world.

3. Do not extract the ZIP file.

4. Launch the world or run:

   ```
   /reload
   ```

5. Verify installation by running:

   ```
   /datapack list
   ```

6. Confirm that **SBTR.zip** appears in the list of enabled datapacks.
